## 核心观点

## 三套核心"芯片"

### 1. 节奏芯片：15字呼吸频率

| 特点 | 说明 |
|------|------|
| 平均句长 | 仅15字 |
| 区间 | 85%句子在11-25字 |
| 效果 | 移动端最舒适的"一口气"长度 |

**线性推进：** 从不纠结过去，始终强调"先做起来"、"越早看清"

### 2. 价值观芯片：七维意识形态

| 维度 | 特点 |
|------|------|
| 精英逻辑 | "赚到钱 = 认知到位" |
| 硬核行动 | 行动力9分，"不懂就干" |
| 克制情感 | 情感浓度仅3分，用"扎心"共情 |

### 3. 修辞芯片：硬核对比算法

- 60%内容基于"对比论证"
- 常用结构："不是X，而是Y"
- 效果：迅速制造认知落差
