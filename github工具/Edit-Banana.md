# Edit Banana - AI 图表可编辑工具

> 原文来源：https://x.com/berryxia/status/2030647797586993304
> 作者：Berryxia.AI

---

## 是什么

**把 AI 生成的静态图片一键转换成可编辑格式的工具。**

今日登上 GitHub 热榜，收获 **2800+ Stars**！

---

## 能做什么

把以下类型的**死图**一键秒变**可编辑**：

- AI 生成的流程图
- 架构图
- PDF 统计图
- 公式图
- 任何截图

---

## 支持导出格式

| 格式 | 说明 |
|------|------|
| **DrawIO** | 可编辑的流程图 |
| **SVG** | 矢量图 |
| **PPTX** | PowerPoint 幻灯片 |

---

## 技术原理

- **SAM3** - 精准分割图像中的各个元素
- **本地 OCR** - 识别文字
- **多模态 LLM** - 还原颜色、箭头、层级、LaTeX 公式
