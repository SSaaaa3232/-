# 强制规则：智能存储路由

> AI 自动识别内容类型，决定存储位置

## 存储规则

| 内容类型 | 存储位置 |
|----------|----------|
| 文章/推文总结 | `x/已思考/` |
| 读书笔记 | `读书/` |
| 工具介绍 | `github工具/` |
| 学习笔记 | `学习/[分类]/` |
| 项目文档 | `03-项目/[项目名]/` |
| 任务模板 | `templates/` |
| 系统文件 | 根目录 |

## 工作流程

1. 识别内容类型
2. 查询存储规则
3. 构建完整路径
4. 执行存储
5. 告知用户位置

## 工作区边界

所有文件必须在系统目录内，禁止保存到外部路径。
