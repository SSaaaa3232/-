# 读书提问话术与模拟

> 创建时间：2026-03-09

---

## 核心原则

```
不要问"作者说了什么"
要问"作者为什么这么说"
         "我该怎么用"
```

---

## 一、读书提问话术

### 1. 看目录时

| 问题 | 目的 |
|------|------|
| "这本书要解决什么问题？" | 明确目标 |
| "作者的核心观点是什么？"（或：最想表达的3个观点是什么？）| 抓住主线 |
| "结构是怎么组织的？" | 建立框架 |
| "和我已知什么有关？" | 连接旧知识 |

### 2. 读章节时

| 问题             | 目的   |
| -------------- | ---- |
| "这一章的核心概念是什么？" | 抓重点  |
| "作者是怎么论证的？"    | 理解逻辑 |
| "有什么证据/案例支持？"  | 验证观点 |
| "和我之前的理解矛盾吗？"  | 批判思考 |
| "这个观点能用在什么地方？" | 应用迁移 |

### 3. 读完后

| 问题 | 目的 |
|------|------|
| "作者最核心的心智模型是什么？" | 提炼模型 |
| "这个领域专家分歧在哪？" | 找到边界 |
| "我能用一句话总结吗？" | 压缩理解 |
| "和我生活/工作有什么关系？" | 应用实践 |
| "还有什么问题没解决？" | 激发下一步 |

## 二、常见疑问解答

### Q1: 一本书只有一个核心观点吗？

**不一定。** 一本书可能有多个核心观点。

更准确的问题：
- "作者最想表达的3个观点是什么？"
- "这一章/这本书的底层框架是什么？"

刚开始不知道怎么回答很正常，可以：
- 先看目录、序言、结论
- 读完后回来总结
- 不是一次就答对，是慢慢理解的

### Q2: 怎么知道结构怎么组织？

**看目录就知道。**

示例：
```
目录：
├── 第一章 社会认知
│   ├── 归因理论
│   ├── 认知偏差
│   └── 自我概念
├── 第二章 社会影响
│   ├── 从众
│   ├── 服从
│   └── 社会规范
```

看目录 → 问"为什么这样排序？" → 理解作者的逻辑

### Q3: NotebookLM 怎么用？

NotebookLM 可以帮你提问和总结。

**导入形式：**

| 形式 | 说明 |
|------|------|
| PDF | 直接上传 PDF 教材/书 |
| YouTube | 视频链接，自动生成摘要 |
| URL | 网页文章 |
| 音频 | 播客等 |

**用法：**
- 导入书 → 问"这一章的核心观点是什么？"
- 导入视频 → 问"讲座的核心框架是什么？"
- 生成 Quiz 考自己

## 三、模拟学习流程（以心理学为例）

### 第一步：选书
假设选了《社会心理学》- 戴维·迈尔斯

### 第二步：看目录，提问

- "这本书解决什么问题？" → 理解人类在社会中的行为、思考、感受方式
- "核心观点？" → 人的行为受社会环境、他人影响巨大
- "结构怎么组织？" → 自我认知 → 社会影响 → 关系 → 态度 → 群体
- "和我已知什么有关？" → 和《影响力》有重叠

### 第三步：读章节（如"社会认知"）

- "核心概念？" → 归因理论、认知偏差、自我概念
- "怎么论证？" → 实验案例 + 数据 + 理论模型
- "有什么证据？" → 斯坦诺维奇实验、米尔格拉姆服从实验
- "和我之前理解矛盾？" → 以为人很理性，其实认知偏差很严重
- "能用在什么地方？" → 营销、谈判、人际沟通

### 第四步：读完后

- "核心心智模型？" → 人不是理性动物，而是"社会动物 + 认知懒虫"
- "专家分歧在哪？" → 先天 vs 后天、理性 vs 情绪
- "一句话总结？" → 人的行为不是孤立的个体决策，而是深深嵌入社会网络中
- "和我生活什么关系？" → 理解自己为什么会在群体中盲从
- "还有什么没解决？" → 如何在数字时代应用这些理论？


---

## 四、注意事项

- **不是一次就答对**，是慢慢理解的
- **问题是迭代的**，读完后可以回头修改之前的答案
- **结合 NotebookLM**，可以辅助提问和总结
