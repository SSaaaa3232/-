## 核心公式

**期望值 (EV) = Σ (概率 × 收益)**

---

## 1. 期望值 (Expected Value)

任何决策都有可能的 outcomes，每个 outcome 有概率和收益。乘起来相加就是期望值。

**例子：**
抛硬币，正面你赢$150，反面你输$100。
```
EV = (50% × $150) + (50% × -$100) = $75 - $50 = +$25
```
每次赌都赚$25，应该接受！

**问题：**
人对损失比收益敏感约2倍。失去$100的痛苦远比赢得$150更强烈。这叫"损失厌恶"，几乎毁掉所有决策。

**现实例子：**
选项A：稳定工作，年薪$120K
选项B：创业公司，60%概率年赚$250K，40%概率年赚$70K

```
EV of A = $120K（稳定）
EV of B = (0.60 × $350K) + (0.40 × $70K) = $150K + $28K = $178K
```

B价值多$58K，但人们通常选A。

## 2. 基础比率忽略 (Base Rate Neglect)

**例子：**
1000人有1人得病，测试99%准确率。

**直觉：** 阳性=99%得病
**实际：**
- 1000人中1人真的有病，被检出（1真阳性）
- 999人没病，但测试1%假阳性（~10人）
- 11个阳性中只有1个真的

**概率 = 1/11 ≈ 9%，不是99%**

**应用：**
- "我创业想法很棒，看看Uber多成功！" 创业成功率约6%
- "去年这个基金赚40%，我应该买！" 连续3年跑赢市场的基金只有约15%
- "Twitter上这个人预测准，他是天才！" 1万人做随机预测，约100人会完美预测

**结论：** 看到成功故事时，问问基础比率是多少。

---

## 3. 沉没成本谬误 (Sunk Cost Fallacy)

**例子：**
电影票$15，看了30分钟发现是烂片。
- 走还是留？

**经济答案：** 走。$15已经花了，留不留都拿不回来。问题是：接下的90分钟做其他事是否更好？

但几乎所有人都会留。"我花钱买的。"

**更严重的例子：**
- "我在这份工作待了3年了，再待3年吧。" 那5年已经过去了，下个3年怎么过？
- "已经亏50%了，不能卖。" 市场不在乎你买的价格。
- "我们在一起6年了。" 那6年发生了，接下来6年呢？

**修复：** 做任何决定时，假装从零开始。忽略你已经花掉的——时间、金钱、情感。只看未来。如果今天让你选，会选这条路吗？

## 4. 贝叶斯定理 (Bayes' Theorem)

**公式：**
P(信念|新证据) = P(证据|信念) × P(信念) / P(证据)

**人话：**
你的更新信念 = 证据支持你理论的力度 × 你之前多相信这个理论 ÷ 证据本身有多常见

**例子：**
你觉得同事会辞职。没有任何证据前，你说10%概率（大多数人任何月份不辞职）。

然后你发现她更新了LinkedIn。

- 如果她真要辞职，更新LinkedIn概率多大？约70%
- 如果她不打算辞职，更新LinkedIn概率多大？人们经常随便更新，约15%

```
P(辞职|更新LinkedIn) = (0.70 × 0.10) / ((0.70 × 0.10) + (0.15 × 0.90))
= 0.07 / (0.07 + 0.135) = 0.07 / 0.205 ≈ 34%
```

一个证据从10%移到34%。不是90%（过度反应），也不是停在10%（忽视证据）。

**这就是预测市场的运作方式。** Polymarket比CNN更准伊朗袭击，因为几千人不断把微小证据输入价格，每次微调0.5%或2%。

**要点：** 弱弱地持有观点，不断更新。更新幅度应与证据强度成正比。

## 5. 幸存者偏差 (Survivorship Bias)

你只看到赢家，严重高估成功概率。

**例子：**
- Crypto Twitter：人人晒盈利，没人晒亏损
- 餐厅："追随热情开餐厅！" 60%第一年倒闭，80%五年倒闭
- 音乐：每天9万首歌上传Spotify，中位数歌曲只获30次播放

**修复：** 有人给你看成功故事当证据，问分母是多少？多少人试过？多少成功？如果找不到分母，假设成功率很低。

## 6. 凯利公式 (Kelly Criterion)

**公式：**
```
f* = (p × b - q) / b
p = 赢的概率
q = 输的概率 (= 1-p)
b = 每元赌注赢多少
```

**例子：**
你发现有60%赢的概率。赢了翻倍(b=1)。
```
f* = (0.60 × 1 - 0.40) / 1 = 0.20
```
凯利说：押20%。

**但问题是：** 全凯利对人类太激进了。50年实际应用证明：专业赌徒和交易员都用1/4到1/2凯利。

所以：押5-10%，不是20%。

**这适用于一切：**
- 职业：不要辞职全押副业。改成4天工作，1天副业。
- 学习：不要一次学5个技能。选最高EV的，深挖。
- 关系：不要泛泛之交。深度投资4-5个重要的人。

**原则：** 有优势时集中，但不要太过度。留出错的空间。

---

## 总结：6个模型

| 模型 | 作用 |
|------|------|
| **EV** | 告诉你要不要行动 |
| **基础比率** | 让估计扎根现实 |
| **沉没成本** | 告诉你要忽略什么 |
| **贝叶斯** | 告诉你如何更新信念 |
| **幸存者偏差** | 告诉你图景里缺了什么 |
| **凯利** | 告诉你该投多少 |
