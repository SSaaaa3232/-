# 挪威 vs 沙特模式

> 章节来源：[[原文]]

---

## 荷兰病

> 资源繁荣 → 资本涌入 → 汇率升高 → 制造业被挤垮

---

## 挪威模式（成功）

- 发现石油前立法规定产权归属
- 政府吸收80%资源租金
- 1996年设立主权财富基金（现$1.9万亿）
- 每年最多支取4%预期收益
- 扶持Equinor（国有石油公司）
- 结果：避开荷兰病，跨代际财富

---

## 沙特模式（困境）

- 经济高度单一
- 主权财富基金大但未实现转型
- "2030愿景"补作业

---

## 核心区别

| | 挪威 | 沙特 |
|---|---|---|
| 加工能力 | 有 | 无 |
| 制度设计 | 强 | 弱 |
| 人才储备 | 有 | 无 |
